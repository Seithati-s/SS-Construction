# SS-Construction
Website for a construction company

## Overview
This is a static website built with HTML, showcasing services likely related to construction or business (based on imagery). Key pages include home, about, services, contact, and enquiry forms.

## File Structure
Seithati/ ├── index.html # Home page ├── About.html # About us page ├── Services.html # Services page ├── Contact.html # Contact page ├── Enquiry.html # Enquiry form page ├── construction site.jpg ├── Home.jpeg ├── office construction.jpeg └── Assets/ └── home.jpeg


## Quick Start
1. Open `index.html` in any web browser.
2. No build tools or servers required – fully static site.

## Pages
- **Home** (`index.html`): Landing page.
- **About** (`About.html`): Company information.
- **Services** (`Services.html`): List of services.
- **Contact** (`Contact.html`): Contact details.
- **Enquiry** (`Enquiry.html`): Form for inquiries.

## Assets
Images are stored directly or in `Assets/` folder. Ensure paths in HTML match file names/casing.

## Development
- Edit HTML files directly.
- Test by opening `index.html` in browser.
- For production, host on any static file host (e.g., GitHub Pages, Netlify).
